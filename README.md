# IQOD_Misc_18

## DO EVERYTHING MANUALLY

```
array = [[4, 5, 6], [2, 3, 4, 5], [7, 5, 3, 4, 1]]
```
Using the array above as an example, create a program that will add the first and last numbers of each array in the parent array.

<hr>
Example:

```
array = [[2, 3], [4, 3, 2]]
iqod(array)
```
Output:

```
4
6
```
